Phase 2 — Character & Prop (First Film)
  • Higgsfield → Soul Cinema → minimal prompt + Enhancer toggle ON → run batches at 0.5 credits each

  • Build a prop sheet: Nano Banana 2 → multiple angles → save all images

Phase 3 — Scene Generation Loop
  • Fill the Generate tab → hit "Generate & Send to Engine"

  • Run 2-3 generations → pick the strongest output
  • Paste that scene's summary into Continuity field → repeat for next scene
